# cookie-analiser

## Usage (Tested on Mac and Windows)

1. Download code from the repo
2. Open Chrome browser
3. Go to extensions, and enable "dev mode"
4. Click load unpucked/uncompressed
   - for porper functionality allow working in the background
5. Go to selected website and click on a extension icon

## Machine Learning model info

`onnx-files` are predefined files required to run .onnx model properly
